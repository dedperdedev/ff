# Trading Bot UI

Trading bot user interface built with vanilla JavaScript, HTML, and CSS.

## Features

- Trading positions management
- Deposit/Withdrawal functionality (TON, Telegram Stars)
- Referral system
- Daily tasks
- Settings panel
- Responsive design

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript

## Setup

1. Clone the repository
2. Open `index.html` in a web browser

## Project Structure

- `index.html` - Main HTML file
- `styles.css` - All styles
- `script.js` - Application logic
